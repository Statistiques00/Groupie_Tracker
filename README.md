# Groupie Tracker
Run: `go run .`
Open: http://localhost:8080
Optional env: `SPOTIFY_CLIENT_ID`, `SPOTIFY_CLIENT_SECRET`
Flags: `-addr`, `-api`, `-static`, `-templates`, `-spotify-client-id`, `-spotify-client-secret`
API: `/api/artists`, `/api/locations`, `/api/dates`, `/api/relation`, `/api/events`, `/api/spotify/artist?id=...`
